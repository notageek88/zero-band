# ZERO Band

> Когда у всех ноль — у тебя всё работает.

Модульный носимый браслет. Офлайн AI-ассистент + защищённая mesh-сеть + датчики + автоинъектор.

## Что это

ZERO работает всегда — с интернетом, без интернета, без сотовой сети, в тайге, в бункере, на поле боя. Не замена телефону — расширение человека.

## Три режима

- **Облако** — есть интернет → полный AI + синхронизация
- **Телефон** — BT рядом → AI на телефоне, браслет как интерфейс
- **Автономия** — ничего нет → малая модель на чипе + LoRa mesh + GPS

## Модули

| Модуль | Назначение |
|--------|-----------|
| Everyday | Повседневка, уведомления, NFC |
| Outdoor | Охота, туризм, mesh до 100 км |
| Tactical | LoRa frequency hopping, AES-256, dead man switch |
| Medical | ЭКГ, автоинъектор (инсулин, адреналин) |
| Rescue | Iridium спутник, радиация, газ |
