# ZERO Band

> Когда у всех ноль — у тебя всё работает.

**ZERO** — модульный носимый LoRa-браслет с mesh-сетью. Связь без вышек, без подписки, без интернета.

## MVP: ZERO Outdoor

Первый продукт — браслет для охотников, туристов и рыболовов:

- **LoRa mesh** до 15 км в лесу / 100 км открытая местность — без подписки
- **GPS-трекинг** с офлайн-навигацией
- **E-ink дисплей** — 7 дней без зарядки
- **SOS-маяк** — координаты через mesh всем в радиусе
- **Совместимость с Meshtastic** — работает с существующей экосистемой

## Почему это нужно

- 40M+ охотников/рыболовов в РФ — связи в глуши нет
- Garmin inReach стоит $350 + $50/мес подписки
- Meshtastic — для технарей, нет коробочного продукта
- GoTenna — недоступен в РФ

## Стек

| Компонент | Технология |
|-----------|-----------|
| MCU | ESP32-S3 (LILYGO T-Beam Supreme) |
| LoRa | Semtech SX1262 (868 MHz) |
| GPS | u-blox NEO-M9N |
| RTOS | ESP-IDF v5.x |
| Mesh-протокол | Meshtastic (совместимый) |
| Дисплей | E-ink / SSD1306 OLED (прототип) |
| Корпус | PETG 3D-печать (прототип) |

## Быстрый старт (прошивка)

```bash
cd firmware
idf.py set-target esp32s3
idf.py build
idf.py flash monitor
```
