# ZERO — Pitch Deck v1.0
> One-pager для инвесторов и партнёров | 2026-03-22

---

## 🎯 Проблема

Смартфон бесполезен без сети. Спутниковые трекеры стоят $50/мес в подписках.
Офлайн AI есть только на ноутбуках. Медицинские устройства — узкоспециализированные.

**В критический момент человек остаётся один.**

---

## 💡 Решение

**ZERO** — модульный носимый браслет.

- Офлайн AI на руке (работает без интернета)
- Mesh-сеть до 100 км без вышек и без подписки
- Датчики здоровья + автоинъектор (Medical модуль)
- Три режима: Облако → Телефон → Автономия (переключение автоматически)

**Когда у всех ноль — у тебя всё работает.**

## 🔧 Технология

- **MCU:** ESP32-S3 (240 MHz, 8MB PSRAM)
- **Связь:** LoRa SX1262 (868 MHz, до 100 км open space)
- **GPS:** u-blox NEO-M9N (multi-band, 10Hz)
- **AI:** Offline LLM (Phi-3 Mini на телефоне + TFLite Micro на чипе)
- **Совместимость:** Meshtastic protocol

Прошивка: open-source (ESP-IDF). Референс: LILYGO T-Beam Supreme.

## 💰 Бизнес-модель

**B2C (основной):** прямые продажи через сайт + маркетплейсы  
**B2B:** охотхозяйства, турбазы, корпоративный аутдор (комплект Node + Outdoor)  
**B2G:** МЧС, поисково-спасательные отряды (Rescue модуль)  
**Сервис:** карты, базы знаний для офлайн AI (без подписки, разовая загрузка)

**Unit economics:**
- Себестоимость Outdoor: ~$45-60 (ODM Китай)
- Цена продажи: ~$200 / 18,000 руб
- Маржа: 3-4x
