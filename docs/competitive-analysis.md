## Рынок

Рынок survivability-гаджетов и тактических носимых устройств оценивается в $4.2B (2025) с ростом 12% в год. Основные драйверы: рост интереса к автономности после пандемии, вооружённые конфликты, природные катастрофы.

## Конкуренты

### 1. Apple Watch Ultra 2
- **Цена:** $799
- **Сильные стороны:** экосистема Apple, дизайн, здоровье (ЭКГ, SpO2), Crash Detection
- **Слабые стороны:**
  - Нет LoRa / mesh-сети
  - Требует iPhone и интернет
  - Полностью бесполезен при отсутствии связи
  - Нет офлайн AI
- **Аудитория:** mainstream, фитнес, городской пользователь

### 2. Garmin inReach Mini 2
- **Цена:** $350 устройство + $15-65/мес подписка (спутниковая связь Iridium)
- **Сильные стороны:** реальная спутниковая связь (Iridium), SOS кнопка, трекинг
- **Слабые стороны:**
  - Подписка обязательна — без неё мёртвый кирпич
  - Нет AI
  - Нет mesh между устройствами
  - Нет медицинских функций
  - Задержка сообщений 10-20 мин
- **Аудитория:** туристы-экстремалы, охотники с большим бюджетом

### 3. GoTenna Pro X2
- **Цена:** $699 за 2 устройства
- **Сильные стороны:** LoRa mesh без подписки, шифрование
- **Слабые стороны:**
  - Только связь — никаких других функций
  - Не носимое (отдельное устройство)
  - Нет AI, нет GPS треккинга, нет датчиков
  - Американский рынок, нет поддержки РФ
- **Аудитория:** военные/тактические, экстренные службы США

### 4. Meshtastic (DIY)
- **Цена:** $25-60 за T-Beam или Heltec устройство
- **Сильные стороны:** бесплатный open-source, огромное сообщество, LoRa mesh
- **Слабые стороны:**
  - DIY — нет коробочного продукта
  - Нет UI/UX
  - Нет AI
  - Нет медицинских датчиков
  - Требует технической компетенции
- **Аудитория:** технари, радиолюбители, разработчики

### 5. Project N.O.M.A.D. (open source, 2026-03-21)
- **GitHub:** github.com/Crosstalk-Solutions/project-nomad (7.7k stars за 1 день)
- **Цена:** бесплатно (open source, требует своё железо)
- **Сильные стороны:** офлайн AI (Ollama), Wikipedia, карты, Docker-based
- **Слабые стороны:**
  - Серверная/десктопная архитектура — не носимое
  - Нет LoRa/mesh
  - Нет датчиков здоровья
  - Требует Raspberry Pi или ноутбук
- **Аудитория:** технари, survival-энтузиасты, prepper-сообщество
- **Значение для ZERO:** подтверждает огромный спрос на офлайн AI (7.7k stars за 1 день = viral)

### 6. Omnipod 5 (автоинъектор инсулина)
- **Цена:** $3,000+/год (расходники)
- **Сильные стороны:** автоматическое дозирование, Bluetooth к телефону, FDA approved
- **Слабые стороны:**
  - Только инсулин — узкая специализация
  - Требует постоянный Bluetooth и смартфон
  - Очень дорого
  - Нет офлайн режима
- **Аудитория:** диабетики
- **Значение для ZERO:** Medical модуль ZERO закрывает более широкий сегмент

## Уникальное позиционирование ZERO

**"Когда у всех ноль — у тебя всё работает"**

ZERO — единственное устройство, которое объединяет:
1. **Mesh P2P без подписки** (как Meshtastic, но в браслете)
2. **Офлайн AI** (как N.O.M.A.D., но носимое)
3. **Медицинские функции** (датчики + автоинъектор)
4. **Модульность** (один браслет → 6 сценариев)
5. **Доступная цена** для РФ рынка

Никто из конкурентов не имеет всё это одновременно.

## Выводы

1. N.O.M.A.D. — прямое подтверждение рынка офлайн AI (viral за 1 день) → время действовать сейчас
2. Meshtastic-сообщество = готовая аудитория, знакома с технологией → партнёрство/community
3. Garmin inReach = ценовой якорь, подписка = боль пользователя → атаковать этот pain point
4. Apple Watch Ultra = aspirational конкурент, не технический
5. Никто не работает на РФ рынке активно → окно возможностей открыто
