## Суть продукта

**ZERO** — модульный носимый браслет. Офлайн AI-ассистент + защищённая mesh-сеть + датчики + автоинъектор.

**Главная идея:** когда у всех ноль — сеть упала, телефон мёртв, интернет вырублен — ZERO работает. Ты не один даже когда один.

**Не замена телефону — расширение человека.**

**Этимология названия:** ZERO от итальянского "zero", от арабского "صِفر" (сифр) — пустота, точка отсчёта. Когда у всех ноль связи — у тебя всё работает.

## Три режима работы (автоматическое переключение)

| Режим | Условие | AI | Связь |
|-------|---------|----|----|
| **Облако** | Есть интернет | Полный Claude/GPT | Все каналы |
| **Телефон** | BT с телефоном | Llama/Phi на телефоне | BT + LoRa |
| **Автономия** | Ничего нет | Малая модель на чипе | LoRa mesh + GPS |

Переключение прозрачное — пользователь не замечает.

## Модульная архитектура

### Core Unit (всегда на руке)
- ESP32-S3 (Wi-Fi + BT + USB)
- E-ink дисплей (батарея живёт в 5-10 раз дольше LCD)
- Базовый аккумулятор 400mAh
- BLE + акселерометр + барометр
- Стандартный разъём с обеих сторон (промышленный, влагозащита)
- Замена модуля за 3 секунды без инструментов

### Линейка модулей

**ZERO Everyday** — только Core
- Повседневные уведомления, здоровье, навигация, NFC оплата

**ZERO Outdoor** — природа, охота, туризм
- LoRa дальний радиус + усиленный GPS + барометр + гибкая солнечная панель
- Mesh с другими носителями до 15 км в лесу / 100 км открытая

**ZERO Tactical** — военные, ЧВК, силовые
- LoRa frequency hopping (защита от глушения)
- AES-256 шифрование
- Ночной режим (дисплей не светит)
- MOLLE-совместимый ремешок
- Кнопки работают в перчатках
- Dead man switch (нет движения 10 мин → автосигнал)

**ZERO Medical** — медики, диабетики, хронические болезни
- ЭКГ электроды + SpO2 + глюкозометр неинвазивный
- Интеграция с CGM (Dexcom G7, FreeStyle Libre) по BT
- **Автоинъектор** (подробнее ниже)

**ZERO Rescue** — МЧС, спасатели
- Спутниковый канал Iridium (SOS в любой точке планеты)
- Радиационный датчик Гейгера
- Газоанализатор (CO, CO2, метан, H2S)
- Стробоскоп 120 дБ для обнаружения
- Яркий ИК-маркер

**ZERO CBRN** — химическая/биологическая/радиационная защита
- Счётчик Гейгера-Мюллера
- MEMS сенсоры химических агентов (зарин, иприт)
- Интеграция с защитным костюмом

### Мини-модули (боковые слоты)
- **Тепловизор** FLIR Lepton (размер монеты) — видишь в темноте
- **Лазерный дальномер** — охота, строительство, тактика
- **Камера 2MP** — фиксация с первой перспективы всегда
- **UV-датчик** — длительные выходы на солнце
- **NFC расширенный** — цифровой ключ, медкарта, оплата

## Автоинъектор (революционная функция)

Микрокартридж под кожу запястья. Меняется раз в день/неделю. Игла 0.3мм — практически безболезненно. Управляется браслетом.

### Применения:

**Инсулин (диабетики)**
- Закрытая петля: CGM датчик → AI рассчитывает дозу → автоинъекция
- Artificial pancreas на запястье
- Рынок: 500 млн диабетиков, текущие решения (Omnipod) $3,000+/год

**Адреналин (аллергики, военные)**
- Одна кнопка на запястье вместо укола в бедро двумя руками
- Авто при анафилактическом шоке (датчики зафиксировали → инъекция)
- Для военных: потеря сознания → браслет вводит адреналин сам

**Обезболивающее (тактическая медицина)**
- Кетамин, трамадол — полевая медицина
- Авто по порогу боли (HRV + кожно-гальваническая реакция)
- Командир через mesh даёт команду → браслет вводит дозу

**Налоксон (передозировка)**
- Детектирует паттерн: SpO2 падает + пульс меняется + нет движения
- Авто-инъекция без участия человека — спасает жизнь
- Рынок США: $10B программы борьбы с опиоидным кризисом

**Ноотропы/кофеин**
- Программируемое микродозирование по расписанию
- Фокус-режим без кофе

### Биометрия

### Окружающая среда

- Радиационный датчик (в Rescue/CBRN)
- Газоанализатор CO, CO2, метан, H2S (в Rescue)
- УФ-индекс
- MEMS хим.агенты (в CBRN)

## Связь (многоуровневая)

| Канал | Радиус | Назначение |
|-------|--------|-----------|
| LoRa 868 MHz | 15 км лес / 100 км открытая | Основной mesh |
| BLE 2.4 GHz | 50-100 м | Телефон, наушники |
| Wi-Fi | Зона покрытия | Синхронизация, обновления |
| LoRa 433 MHz | резервный | Другая частота |
| Iridium спутник | Весь мир | SOS (Rescue модуль) |

**Frequency hopping** для Tactical — защита от глушения.

## Автономность

| Режим | Время |
|-------|-------|
| Обычное использование | 7 дней |
| Режим маяка (SOS пинг раз в мин) | 30 дней |
| С солнечной панелью | Бесконечно |

### Решения по энергопотреблению:
- E-ink дисплей — обновляется только при изменении, не потребляет в статике
- Wake on motion — поднял руку → экран включился, опустил → выключился
- Адаптивный LoRa — активный приём ↔ beacon раз в минуту
- Съёмный аккумулятор 18650 — замена за 5 секунд
- Qi беспроводная зарядка + USB-C
- Гибкая солнечная панель на внешней стороне (Outdoor модуль)
- Термоэлектрический генератор (разница температур тело/воздух)

## AI архитектура

### На браслете (всегда):
- Wake word детектор "Зеро, помоги" — TensorFlow Lite Micro, 50KB
- Базовые команды: SOS, метка точки, сообщение, здоровье
- Офлайн база знаний (первая помощь, съедобные растения, навигация) — векторный поиск, не LLM

### Через телефон (BT, авто):
- Llama 3.2 3B или Phi-3 Mini — полноценный разговорный AI
- Whisper для голосового ввода
- Сложные вопросы, планирование

### Через интернет:
- Полный Claude/GPT через API
- Синхронизация памяти, обновление баз знаний

## ZERO Node (отдельный продукт)

Размер спичечного коробка. Питание USB или солнечная панель.
Вешается на дерево, столб, крышу — ретранслирует mesh.

- Один Node: покрытие 30-50 км вокруг
- 10 Node по области: готовая инфраструктура без вышек связи

**Продаётся:** охотхозяйствам, МЧС-базам, муниципалитетам, туристическим базам.
Решает проблему курицы и яйца (нет пользователей → нет mesh → нет смысла в браслете).

## Технический стек

### Железо
- MCU: ESP32-S3
- LoRa: Semtech SX1262
- GPS: u-blox NEO-M9N
- IMU: ICM-42688-P
- Дисплей: E-ink SSD1306 / GC9A01 круглый
- Аккумулятор: TP4056 + Li-Po 400-800mAh
- Зарядка: Qi + USB-C

### Прошивка
- RTOS: Zephyr (для носимых) или ESP-IDF (быстрее старт)
- AI на чипе: TensorFlow Lite Micro
- Mesh: свой протокол поверх LoRa (Meshtastic как референс)

### AI стек
- На устройстве: TFLite Micro (wake word + команды)
- На телефоне: llama.cpp + Whisper
- Облако: Anthropic API / OpenAI API

### 3D проектирование
- FreeCAD (открытый)
- Печать: PETG (гибче и прочнее PLA для носимых)
