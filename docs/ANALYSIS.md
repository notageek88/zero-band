## Сильные стороны

### 1. Реальная боль
- 40M+ охотников/рыболовов в РФ без связи в глуши
- Garmin inReach = $350 + $50/мес подписки — дорого и привязка
- GoTenna недоступен в РФ
- Ниша пуста для коробочного продукта

### 2. Отличный timing
- Project N.O.M.A.D. набрал 7.7k stars за 1 день — подтверждение спроса на офлайн-технологии
- Meshtastic-сообщество растёт экспоненциально
- Тренд на автономность после 2022 года в РФ

### 3. Модульность = множитель рынка
- Один Core Unit → 5+ аудиторий
- Каждый модуль — отдельный revenue stream
- Открытый стандарт разъёма → экосистема сторонних производителей

### 4. Дешёвый прототип
- $138 на T-Beam Supreme — рабочий mesh за 1 месяц
- Не нужны $100k+ на R&D первой итерации

### 5. Здоровая unit economics
- Себестоимость ODM: ~$45-60
- Цена продажи: ~$200 / 18,000 руб
- Маржа: 3-4x

## Критические риски

### 1. Scope creep (ГЛАВНАЯ УГРОЗА)
Концепт пытается быть всем: mesh-рация, AI-ассистент, медприбор, тактика, инъектор, тепловизор, дозиметр. Ни одна hardware-компания не запускала 6+ модулей одновременно.

**Примеры фокуса у успешных:** Fitbit начинал с шагомера. Apple Watch — с уведомлений. Garmin — с GPS-навигатора.

### 2. Автоинъектор = регуляторный ад
- Медицинское изделие класса III
- Регистрация Росздравнадзора: 12-24 мес, $50-200k
- Клинические испытания обязательны
- FDA 510(k) для международного рынка
- ISO 13485 система качества
- Omnipod тратит $200M+/год на R&D одного инъектора

### 3. Hardware = не соло-проект
Необходимая команда:
- Инженер-электронщик (PCB, RF, power management)
- Инженер-механик (корпус, IP68)
- Embedded-разработчик (RTOS, RF-стек)
- Сертификация (FCC, CE, ФСТЭК)

### 4. Энергопотребление
Заявлено 7 дней, но реально при активном использовании (LoRa mesh + GPS continuous + BLE):
- GPS: ~30mA постоянно
- LoRa mesh с frequency hopping: высокое потребление
- 400mAh аккумулятор → реально 1-2 дня
- Нужна оптимизация: duty cycling, adaptive GPS

### 5. "Офлайн AI" — маркетинговая натяжка
- TFLite Micro на ESP32 = wake word detection (уровень "OK Google")
- Полноценный LLM (Phi-3 Mini) требует 4GB RAM
- ESP32-S3 имеет 8MB PSRAM — разница в 500x
- На устройстве реально: предзаписанные команды, не AI-ассистент

### 6. Конкуренция
- Apple Watch Ultra 3 может добавить спутниковый mesh
- Meshtastic-сообщество может выпустить носимый форм-фактор
- Китайские клоны через 12-18 мес

## Рекомендации

### 1. Убить 80% scope — запускать только ZERO Outdoor
Один модуль. Одна аудитория (охотники). Одна killer-фича: LoRa mesh без подписки в форм-факторе браслета. Без AI, без инъектора, без тактики, без медицины. Всё остальное — roadmap на 2-3 года.

### 2. Первый шаг — валидация спроса, не прошивка
- Telegram-канал + демо-видео с T-Beam в лесу
- Лендинг + предзаказы (СБП/карта)
- 200 предзаказов = go, меньше = pivot

### 3. Meshtastic — фундамент, не конкурент
Не писать свой mesh-протокол. Использовать Meshtastic firmware + красивый корпус + UX + мобильное приложение. Продукт — это user experience, не технология.

### 4. Автоинъектор = отдельная компания
Отдельный стартап, отдельное финансирование ($2-5M seed), отдельная команда медтех-инженеров, отдельная сертификация. Не мешать с браслетом.

### 5. Искать сооснователя с hardware-опытом
Embedded/RF инженер с опытом серийного производства. Без него проект останется на уровне T-Beam с напечатанным корпусом.
