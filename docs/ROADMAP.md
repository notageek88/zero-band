## Фаза 1: Прототип на T-Beam — ~$150, 4-6 недель

| Задача | Описание |
|--------|----------|
| Купить T-Beam Supreme x2 | Aliexpress, LILYGO Official Store |
| Meshtastic firmware | Залить стандартную прошивку, проверить mesh |
| Тест дальности | 2 устройства, замер: лес, город, открытая местность |
| GPS-трекинг | Запись маршрута, отображение на карте |
| SOS-функция | Кнопка → broadcast координат через mesh |
| Демо-видео | 3-5 мин, реальные условия (лес/поле) |

**Результат:** работающий mesh между 2 устройствами + видео-proof.

## Фаза 3: Кастомная PCB — ~$500-1000, 6-8 недель

*Только при 200+ предзаказах*

| Задача | Описание |
|--------|----------|
| KiCad схемотехника | ESP32-S3 + SX1262 + GPS + дисплей на одной плате |
| Заказ PCB | JLCPCB, 5-10 плат ($10-20) |
| Пайка и тест | Проверка всех узлов |
| Корпус v1 | FreeCAD → 3D-печать PETG |
| Форм-фактор браслета | Итерации, тест на руке |

**Результат:** собственная плата в форм-факторе браслета.
